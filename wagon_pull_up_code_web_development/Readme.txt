How to run the Wagon Pull Up (Vehicle Parking Management System) Project

1. Download the zip file
2. Extract the file and copy the wpu folder
3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/HTML)
4. Open PHPMyAdmin (http://localhost/phpmyadmin)
5. Create a database with name wpudb
6. Import wpudb.sql file(given inside the zip package in  sql file folder)
7.Run the script http://localhost/wpu (frontend)
